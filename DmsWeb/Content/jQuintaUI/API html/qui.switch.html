jQuintaUI Switch

jQuery widžeti kodirani prvenstveno za potrebe projekata. Zahtevaju jQuery 1.7+ i jQuery.ensureVisible koji je već uključen u kod vidžeta.

Opcije

>>> $('#id').quiSwitch({ opcije });
margin: '3px 0 0 0'
Margina kontrole. Standardni margin CSS property.
on: false
Da li je vidžet inicijalno uključen ili isključen.
height: 18
Visina vidžeta.
width: 60
Širina vidžeta.
display: 'inline-block'
Prikaz kontrole. Standardni display CSS property.
verticalAlign: 'top'
Vertikalno poravnanje kontrole. Standardni vertical-align CSS property.
onBackColor: '#9c0'
Pozadinska boja ON dela vidžeta. Standardni background-color CSS property.
offBackColor: '#ccc'
Pozadinska boja OFF dela vidžeta. Standardni background-color CSS property.
onText: 'ON'
Tekst ON dela vidžeta.
offText: 'OFF'
Tekst OFF dela vidžeta.
onTextColor: '#444'
Boja teksta ON dela vidžeta. Standardni color CSS property.
offTextColor: '#444'
Boja teksta OFF dela vidžeta. Standardni color CSS property.
tickColor: 'gray'
Pozadinska boja izmedju ON i OFF dela vidžeta. Standardni background-color CSS property.

Metode (samo podvučene metode dozvoljavaju ulančavanje!)

>>> $('#id').quiSwitch('nazivMetode', argument1, argument2, ...);

toggle

$('#id').quiSwitch('toggle')
Setuje vrednost suprotno od trenutne.

setOn

$('#id').quiSwitch('setOn', true/false)
Setuje prosledjenu vrednost.

isOn

$('#id').quiSwitch('isOn')
Proverava da li je vidžet na ON ili OFF.
