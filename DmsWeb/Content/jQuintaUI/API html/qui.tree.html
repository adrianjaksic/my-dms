jQuintaUI Tree

jQuery widžeti kodirani prvenstveno za potrebe projekata. Zahtevaju jQuery 1.7+ i jQuery.ensureVisible koji je već uključen u kod vidžeta.

Opcije

>>> $('#id').quiTree({ opcije });
array: []
Niz elemenata koji će se adaptirati za prikaz u obliku stabla.
idProp: 'id'
Naziv identifikacionog pripertija elementa niza.
idParentProp: 'parentid'
Naziv identifikacionog pripertija roditelja elementa niza.
labelProp: ''
Naziv prikaznog pripertija elementa niza.
width: 0
Visina kontrole. Ako je 0, kontrola je samoproširiva.
height: 0
Širina kontrole. Ako je 0, kontrola je samoproširiva.
rootValue: 0
Vrednost za 'idParentProp' koja označava da element treba da se nadje u root-u drveta.
dblClickOpensNode: true
Da li dblClick otvara čvor ili ne.

Metode (samo podvučene metode dozvoljavaju ulančavanje!)

>>> $('#id').quiTree('nazivMetode', argument1, argument2, ...);

getSelectedItem

$('#id').quiTree('getSelectedItem')
Vraća selektovani element.

getSelectedItemDataProperty

$('#id').quiTree('getSelectedItemDataProperty', 'prop')
Vraća property data objekata selektovanog elementa, koji je definisan preko PROP stringa / undefined.

getSelectedItemId

$('#id').quiTree('getSelectedItemId')
Vraća ID selektovanog elementa.

addItem

$('#id').quiTree('addItem', {item}, addType)
Dodaje elemet u stablo.
addType 1: u root
addType 2: ispod selektovanog
addType 3: ispod zadatog parentId-a u {item} objektu

removeSelectedItem

$('#id').quiTree('removeSelectedItem')
Uklanja selektovani element.

Događaji

>>> $('#id').on('event', function ( ... ) { ... });

select

$('#id').on('select', function (event, selectedItem) { })
Trigeruje se nakon selekcije elementa. Događaj pruža standardni event objekat, kao i selectedItem objekat, isti onaj koji bi se dobio metodom 'getSelectedItem'.
