jQuintaUI ComboBox

jQuery widžeti kodirani prvenstveno za potrebe projekata. Zahtevaju jQuery 1.7+ i jQuery.ensureVisible koji je već uključen u kod vidžeta.

Opcije

>>> $('#id').quiComboBox({ opcije });
height: 22
Visina kontrole
width: 198
Širina kontrole
listHeight: 180
Visina padajuće liste
listWidth: 0
Širina padajuće liste
showX: true
Prikaži dugmence za poništavanje selekcije.
showSearch: true
Prikaži polje za pretragu i filtriranje elemenata.
disabled: false
Isključi vidžet.
openUp: false
Ako se prosledi true, vidžet se otvara na gore umesto na dole.
searchType: 'filter'
Način pretrage elemenata u listi.
'filter' - filtritaju se elementi koji sadrže otkucani substring, markira se prvi
'filter2' - isto kao i 'filter' s razlikom da se ne pravi razlika između čćžš i czs
'filterKonto' - isto kao i 'filter' ali prilagodjeno za pretragu konta
'firstHit' - pretražuju se elementi samo do prvog koji sadrži otkucani substring i on se i markira
NOTE: ako se nalazi više od 300 elemenata u list, automatski se pretraga svičuje na 'firstHit' jer je ova mnogo manje zahtevna za procesor
zIndex: ''
Ako se navede, vidžetu se setuje z-index. (Korisno ako otvorena lista prelazi preko nekih elementa koji imaju z-index)
timeout: 300
Zadrška pre nego što krene pretraga nakon kucanja, u milisekundama.
items: []
Niz elemenata za prikaz. Podržava dve strukture:
1. običan niz stringova ili brojeva: 
[ "Jedan", "Dva", "Tri", ... ]
2. niz pravilno struktuiranih objekata koji imaju polja label (ono što će se prikazati) i data (bilo kakav JS objekat): 
[
                                { label: "Naziv", data: podaci },
                                { label: "Naziv", data: podaci },
                                { label: "Naziv", data: podaci },
                                ...
                            ]

Metode (samo podvučene metode dozvoljavaju ulančavanje!)

>>> $('#id').quiComboBox('nazivMetode', argument1, argument2, ...);

getSelectedItem

$('#id').quiComboBox('getSelectedItem')
Vraća selektovani element u obliku { label: 'string', data: object/undefined }.

getSelectedItemData

$('#id').quiComboBox('getSelectedItemData')
Vraća data objekat selektovanog elementa / undefined.

getSelectedItemDataProperty

$('#id').quiComboBox('getSelectedItemDataProperty', 'prop')
Vraća property data objekata selektovanog elementa, koji je definisan preko PROP stringa / undefined.

getSelectedItemLabel

$('#id').quiComboBox('getSelectedItemLabel')
Vraća labelu selektovanog elementa / undefined.

getSelectedIndex

$('#id').quiComboBox('getSelectedIndex')
Vraća redni broj/index selektovanog elementa ili undefined ako ništa nije selektovano.

getItems

$('#id').quiComboBox('getItems')
Vraća niz svih elemenata, gde je pojedinačan objekat niza oblika { label: 'string', data: object/undefined }, kao kod ITEMS opcije.

clearItems

$('#id').quiComboBox('clearItems')
Sklanja sve elemente iz kontrole i deselektuje je.

clearSelection

$('#id').quiComboBox('clearSelection')
Sklanja trenutno selektovani element.

clearSelectionNoTrigger

$('#id').quiComboBox('clearSelectionNoTrigger')
Sklanja trenutno selektovani element. NE TRIGERUJE SELECT DOGADjAJ.

setItems

$('#id').quiComboBox('setItems', [itemsArray])
Popunjava combobox elementima iz prosleđenog niza. Struktura niza je kao kod ITEMS opcije u konstruktoru.

setItemsFromSelect

$('#id').quiComboBox('setItemsFromSelect', '#selectId')
Popunjava combobox elementima iz Select html taga, gde se text Option taga prepoznaje kao label polje u strukturi ITEMS, a value atribut (ako postoji) kao data polje ITEMS strukure. Id Select taga se prosledjuje sa #.

setItemsFromBinding

$('#id').quiComboBox('setItemsFromBinding', [objectArray], 'labelProperty', 'dataProperty')
Popunjava combobox elementima iz nekog drugog niza. Potrebno je proslediti taj niz, a zatim navesti naziv propertija objekta niza koji će predstavljati labelu u listi i naziv propertija koji će se zapamtiti kao data elementa.

setItemsFromBinding2

$('#id').quiComboBox('setItemsFromBinding2', [objectArray], 'labelProperty')
Popunjava combobox elementima iz nekog drugog niza. Potrebno je proslediti taj niz, a zatim navesti naziv propertija objekta niza koji će predstavljati labelu u listi. Ceo pojedinačan objekat niza se pamti u data.

selectItemByIndex

$('#id').quiComboBox('selectItemByIndex', number)
Selektuje element za zadati 'zero' redni broj/index.

selectItemByIndexNoTrigger

$('#id').quiComboBox('selectItemByIndexNoTrigger', number)
Selektuje element za zadati 'zero' redni broj/index. NE TRIGERUJE SELECT DOGADjAJ.

selectItemByLabel

$('#id').quiComboBox('selectItemByLabel', 'string')
Selektuje element za zadatu labelu.

selectItemByLabelNoTrigger

$('#id').quiComboBox('selectItemByLabelNoTrigger', 'string')
Selektuje element za zadatu labelu. NE TRIGERUJE SELECT DOGADjAJ.

selectItemByData

$('#id').quiComboBox('selectItemByData', 'string'/number/(object) )
Selektuje element gledajuci njegov data objekat. Poželjno je proslediti samo prost tip, string ili broj.

selectItemByDataNoTrigger

$('#id').quiComboBox('selectItemByDataNoTrigger', 'string'/number/(object) )
Selektuje element gledajuci njegov data objekat. Poželjno je proslediti samo prost tip, string ili broj. NE TRIGERUJE SELECT DOGADjAJ.

selectItemByDataProperty

$('#id').quiComboBox('selectItemByDataProperty', 'prop', value)
Selektuje element gledajuci propertije data objekta. PROP proslediti kao string, a VALUE je prost tip.

selectItemByDataPropertyNoTrigger

$('#id').quiComboBox('selectItemByDataPropertyNoTrigger', 'prop', value)
Selektuje element gledajuci propertije data objekta. PROP proslediti kao string, a VALUE je prost tip. NE TRIGERUJE SELECT DOGADjAJ.

enable

$('#id').quiComboBox('enable', true/false )
Uključuje (true) ili isključuje (false) vidžet.

isEnabled

$('#id').quiComboBox('isEnabled' )
Proverava da li je vidžet uključen il nije.

setOpenUp

$('#id').quiComboBox('setOpenUp', true/false)
Metoda za menjanje načina otvaranja liste, na gore (true) ili difoltno na dole (false).

Događaji

>>> $('#id').on('event', function ( ... ) { ... });

select

$('#id').on('select', function (event, selectedItem) { })
Trigeruje se nakon selekcije elementa. Događaj pruža standardni event objekat, kao i selectedItem objekat, isti onaj koji bi se dobio metodom 'getSelectedItem'.
