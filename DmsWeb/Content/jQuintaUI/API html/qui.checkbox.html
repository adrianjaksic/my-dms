jQuintaUI CheckBox

jQuery widžeti kodirani prvenstveno za potrebe projekata. Zahtevaju jQuery 1.7+ i jQuery.ensureVisible koji je već uključen u kod vidžeta.

Opcije

>>> $('#id').quiCheckBox({ opcije });
height: 18
Visina i širina kontrole.
margin: '3px 0 0 0'
Margina kontrole. Standardni margin CSS property.
display: 'inline-block'
Prikaz kontrole. Standardni display CSS property.
verticalAlign: 'top'
Vertikalno poravnanje kontrole. Standardni vertical-align CSS property.
checked: false
Inicijalno stanje.

Metode (samo podvučene metode dozvoljavaju ulančavanje!)

>>> $('#id').quiCheckBox('nazivMetode', argument1, argument2, ...);

toggle

$('#id').quiCheckBox('toggle')
Menja stanje kontrole u suprotno od trenutnog.

toggleNoTrigger

$('#id').quiCheckBox('toggleNoTrigger')
Menja stanje kontrole u suprotno od trenutnog. NE TRIGERUJE DOGADjAJ.

setChecked

$('#id').quiCheckBox('setChecked', [true/false])
Setuje stanje kontrole.

setCheckedNoTrigger

$('#id').quiCheckBox('setCheckedNoTrigger', [true/false])
Setuje stanje kontrole. NE TRIGERUJE DOGADjAJ.

isChecked

$('#id').quiCheckBox('isChecked')
Vraća stanje kontrole, vrednosti true/false.

Događaji

>>> $('#id').on('event', function ( ... ) { ... });

change

$('#id').on('change', function (event, check) { })
Trigeruje se nakon promene stanja kontrole. Događaj pruža standardni event objekat, kao i check objekat koji je vrednosti true/false.
