jQuintaUI Date

jQuery widžeti kodirani prvenstveno za potrebe projekata. Zahtevaju jQuery 1.7+ i jQuery.ensureVisible koji je već uključen u kod vidžeta.

Opcije

>>> $('#id').quiDate({ opcije });
date: new Date()
Datum kontrole. Ako se ne navede, difoltni je današnji datum.
width: 198
Širina kontrole.
height: 22
Visina kontrole.
calendarHeight: 210
Visina kalendara.
disabled: false
Isključi vidžet.
openUp: false
Ako se prosledi true, vidžet se otvara na gore umesto na dole.
showX: false
Prikaži dugmence za poništavanje datuma.
zIndex: ''
Ako se navede, vidžetu se setuje z-index. (Korisno ako otvoren kalendar prelazi preko nekih elementa koji imaju z-index)

Metode

>>> $('#id').quiDate('nazivMetode', argument1, argument2, ...);

getDate

$('#id').quiDate('getDate')
Vraća selektovani datum u obliku JS Date() objekta.

getParsedDate

$('#id').quiDate('getParsedDate')
Vraća selektovani datum u formatiranom obliku za prikaz (npr: 3.10.2010.).

getJSONDateNoTZ

$('#id').quiDate('getJSONDateNoTZ')
Vraća selektovani datum u JSON obliku bez timezone.

getJSONDateNoTZStringify

$('#id').quiDate('getJSONDateNoTZStringify')
Vraća selektovani datum u JSON obliku bez timezone u prilagodjenom obliku za slanje pojedinacnog datuma serveru.

getJSONDate

$('#id').quiDate('getJSONDate')
Vraća selektovani datum kodiran u JSON string koji je pogodan za slanje ka serverskom delu aplikacije.

setDate

$('#id').quiDate('setDate', date)
Setuje datum vidžeta. Potrebno je proslediti JS Date() objekat.

setSpecialDate

$('#id').quiDate('setSpecialDate', 'date', noTrigger)
Setuje specijalan datum vidžeta. Moguće opcije: 'firstDayOfYear', 'firstDayOfMonth', 'lastDayOfYear', 'lastDayOfMonth'. Ako se za drugi parametar, noTrigger, prosledi true, neće se okinuti setDate događaj.

clearDate

$('#id').quiDate('clearDate', date)
Sklanja datum vidžeta.

setDateNoTrigger

$('#id').quiDate('setDate', date)
Setuje datum vidžeta. Potrebno je proslediti JS Date() objekat. NE TRIGERUJE SETDATE DOGADjAJ.

enable

$('#id').quiDate('enable', true/false )
Uključuje (true) ili isključuje (false) vidžet.

Događaji

>>> $('#id').on('event', function ( ... ) { ... });

setDate

$('#id').on('setDate', function (event, date) { })
Trigeruje se nakon setovanja novog datuma. Događaj pruža standardni event objekat, kao i date objekat, isti onaj koji bi se dobio metodom 'getDate'.
