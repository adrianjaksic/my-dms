jQuintaUI Multiselect

jQuery widžeti kodirani prvenstveno za potrebe projekata. Zahtevaju jQuery 1.7+ i jQuery.ensureVisible koji je već uključen u kod vidžeta.

Opcije

>>> $('#id').quiMultiselect({ opcije });
height: 22
Visina kontrole
width: 198
Širina kontrole
listHeight: 180
Visina padajuće liste
listWidth: 0
Širina padajuće liste
showSearch: true
Prikaži polje za pretragu i filtriranje elemenata.
disabled: false
Isključi vidžet.
openUp: false
Ako se prosledi true, vidžet se otvara na gore umesto na dole.
searchType: 'filter'
Način pretrage elemenata u listi.
'filter' - filtritaju se elementi koji sadrže otkucani substring, markira se prvi
'filter2' - isto kao i 'filter' s razlikom da se ne pravi razlika između čćžš i czs
'filterKonto' - isto kao i 'filter' ali prilagodjeno za pretragu konta
'firstHit' - pretražuju se elementi samo do prvog koji sadrži otkucani substring i on se i markira
NOTE: ako se nalazi više od 300 elemenata u list, automatski se pretraga svičuje na 'firstHit' jer je ova mnogo manje zahtevna za procesor
zIndex: ''
Ako se navede, vidžetu se setuje z-index. (Korisno ako otvorena lista prelazi preko nekih elementa koji imaju z-index)
timeout: 300
Zadrška pre nego što krene pretraga nakon kucanja, u milisekundama.

Metode (samo podvučene metode dozvoljavaju ulančavanje!)

>>> $('#id').quiMultiselect('nazivMetode', argument1, argument2, ...);

getSelectedItems

$('#id').quiMultiselect('getSelectedItems', type)
Vraća selektovane elemente.
type 1: vraca niz objekata koji su selektovani, menja objekat i stavlja true u checkedProp
type 2: default - vraca niz objekata koji su selektovani, ne menja se objekat

getItems

$('#id').quiMultiselect('getItems', type)
Vraća sve elemente.
type 1: vraca niz svih objekata, menja objekat i stavlja true/false u checkedProp
type 2: default - vraca niz svih objekata, ne menja objekat

selectAll

$('#id').quiMultiselect('selectAll')
Selektuje sve elemenete u widgetu.

deselectAll

$('#id').quiMultiselect('deselectAll')
Deselektuje sve elemenete u widgetu.

setItems

$('#id').quiMultiselect('setItems', [objectArray], 'labelProperty', 'checkedProperty')
Popunjava multiselect elementima iz niza. Potrebno je proslediti taj niz, a zatim navesti naziv propertija objekta niza koji će predstavljati labelu u listi i naziv propertija koji će određivati stanje checked/unchecked.

enable

$('#id').quiMultiselect('enable', true/false )
Uključuje (true) ili isključuje (false) vidžet.

isEnabled

$('#id').quiMultiselect('isEnabled' )
Proverava da li je vidžet uključen il nije.

setOpenUp

$('#id').quiMultiselect('setOpenUp', true/false)
Metoda za menjanje načina otvaranja liste, na gore (true) ili difoltno na dole (false).

updateWidgetLabel

$('#id').quiMultiselect('updateWidgetLabel')
Metoda za manuelno poterivanje apdejta labele widgeta.
