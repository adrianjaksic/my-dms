jQuintaUI Progress

jQuery widžeti kodirani prvenstveno za potrebe projekata. Zahtevaju jQuery 1.7+ i jQuery.ensureVisible koji je već uključen u kod vidžeta.

Opcije

>>> $('#id').quiProgress({ opcije });
height: 18
Visina kontrole.
width: 200
Širina kontrole.
step: 0
Korak za koji se čita prilikom pozivanja metode 'stepUp'.
perc: 0
Inicijlana procenat popunjenosti.
margin: '3px 0 0 0'
Margina kontrole. Standardni display CSS property.
display: 'inline-block'
Prikaz kontrole. Standardni display CSS property.

Metode (samo podvučene metode dozvoljavaju ulančavanje!)

>>> $('#id').quiProgress('nazivMetode', argument1, argument2, ...);

set

$('#id').quiProgress('set', percent)
Setuje procenat popunjenosti widgeta. Broj izmedju 0 i 100.

stepUp

$('#id').quiProgress('stepUp')
Inkrementira procenat popunjenosti widgeta za vrednost iz 'step' propertija. Ide do 100.
