jQuintaUI TableFilter

jQuery widžeti kodirani prvenstveno za potrebe projekata. Zahtevaju jQuery 1.7+ i jQuery.ensureVisible koji je već uključen u kod vidžeta.

Opcije

>>> $('#id').quiTableFilter({ opcije });
tbodyId: ''
#Id tbody tabele na koji se odnosi filtriranje.

Događaji

>>> $('#id').on('event', function ( ... ) { ... });

filtered

$('#id').on('filtered', function (event) { })
Trigeruje se nakon izvrsenog filtriranja.

cleared

$('#id').on('cleared', function (event) { })
Trigeruje se nakon izvrsenog ponistavanja filtera.
